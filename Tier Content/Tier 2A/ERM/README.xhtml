  • Schematron

    Collection of constraint rules applicable to this specification. These rules depend on the CVEs being located at the "correct" relative path they are in this package. Many of the dependent specifications have applicable constraint rules; each one is documented separately. Each specification has three main files.

    • xxx_Rules.pdf

      A generated PDF collection of all the constraint rules. They are actually many individual .sch files. This file is a more convenient format to read but is not useful for execution of the rules.

    • xxx_XML.sch

      The top level .sch file that includes all of the others for the particular specification.

    • xxx_XML.xsl

      A pre "compiled" version of the .sch file ready for execution with any XSLT2 capable engine.

    Applicable constraint rules are as follows:
